Buttons

Buttons are created using the .al-btn component class, combined with a color modifier like .al-btn-primary.

Examples

HTML

<!-- Solid Buttons -->
<button class="al-btn al-btn-primary">Primary</button>
<button class="al-btn al-btn-danger">Danger</button>
<button class="al-btn al-btn-success">Success</button>

<!-- Special Buttons -->
<button class="al-btn al-btn-gradient">Gradient</button>
<button class="al-btn al-btn-primary-outline">Outline</button>

<!-- Disabled State -->
<button class="al-btn al-btn-primary" disabled>Disabled</button>

Aces's Lair Custom Language (ALCL)

ALCL is a utility-first CSS framework, just like Tailwind, designed for rapid prototyping with our high-tech aesthetic. Include the script link in your HTML and start building.

Script Link

<link rel="stylesheet" href="http://localhost:5500/aces-lair.css">

Example: Building a Button with ALCL

Instead of using pre-defined components, you can construct elements from low-level utility classes.

HTML

<button class="al-btn al-btn-primary font-orbitron">
  ALCL Button
</button>

Responsive Design

ALCL uses a mobile-first approach. Utilities apply to all screen sizes by default. To apply a utility only on larger screens, prefix it with a breakpoint name.

  • md: (768px and up) - for tablets
  • lg: (1024px and up) - for desktops

Example: Responsive Grid

This grid starts as a single column on mobile, becomes two columns on medium screens, and four columns on large screens.

  <div>...</div>
  <div>...</div>

  <div>...</div>
</div>

Accordion

A vertically stacked set of interactive headings that each contain a title, content snippet, or thumbnail representing a section of content.

Example

This is the content for the first section. It can contain any HTML elements.

This is the content for the second section. Accordions are useful for showing and hiding large amounts of content in a compact space.

HTML

<div class="accordion">
  <div class="accordion-item">
    <button class="accordion-header">Section 1</button>
    <div class="accordion-content">
      <p>Content for section 1...</p>
    </div>
  </div>
  <div class="accordion-item">
    <button class="accordion-header">Section 2</button>
    <div class="accordion-content">
      <p>Content for section 2...</p>
    </div>
  </div>
</div>

JavaScript

const accordionHeaders = document.querySelectorAll('.accordion-header');

accordionHeaders.forEach(header => {
  header.addEventListener('click', () => {
    header.classList.toggle('active');
    const content = header.nextElementSibling;
    if (content.style.maxHeight) {
      content.style.maxHeight = null;
    } else {
      content.style.maxHeight = content.scrollHeight + 'px';
    }
  });
});

Modal

A modal is a dialog box/popup window that is displayed on top of the current page.

Example

HTML

<!-- Trigger Button -->
<button class="al-btn al-btn-primary" data-modal-target="#example-modal">Open Modal</button>

<!-- Modal Structure -->
<div class="modal-overlay" id="example-modal">
  <div class="modal-content">
    <div class="modal-header">
      <h4>Modal Title</h4>
      <button class="modal-close">&times;</button>
    </div>
    <div class="modal-body">
      <p>This is the main content of the modal...</p>
    </div>
  </div>
</div>
